# Agreg_Papi

-> Instalation :
    - Récupérer le dossier et le placer à l'endroit voulu (noter ici : ...) .
    - Faire un raccourci du fichier 'Agreg_Papi.exe'
    - Placer le raccourci dans le dossier : ""
Il se démarera alors à chaque démarrage de l'ordinateur

-> Désinstallation :
    - Aller dans le dossier : ""
    - Supprimer le raccourci 'Agreg_Papi'.
    - retourner au dossier noté plus haut lors de l'installation (le premier) et supprimer le
    dossier Agreg_Papi.
